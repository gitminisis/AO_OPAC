Search tips

Add an asterisk (*) to expand your search results

Add an asterisk (*) to the end of a word to get results that include variations of that word. For example, entering "farm*" will return results that include "farm", "farmers", and "farming".

View results with a digital image

Select the "Digital Media Present" filter after you search.

Reference code (archival advanced search only)

Include a space between the letters and numbers, and include a dash between the numbers. For example, "RG 1-11".
